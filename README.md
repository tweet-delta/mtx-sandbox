# MTX

A small sandbox project for practicing the GitHub pull request workflow.

## What is this?

This repo exsits so that changes can be proposed, reviewed, and merged through
pull requests, just like a real project.

## Getting started

1. Clone the repo
2. Make a change on a branch
3. Open a pull request
4. Review and merge

## Learning Log

- Learned what Claude Code, git, GitHub, branches, commits, and PRs are.
